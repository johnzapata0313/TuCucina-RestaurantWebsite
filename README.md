# Los Arrieros Restaurant 🇨🇴

A responsive restaurant website built for **Los Arrieros**, an authentic Colombian restaurant based in Philadelphia, PA.

## Live Demo

[View Site]([https://johnzapata0313.github.io/TuCucina-RestaurantWebsite](https://tucucina-restaurant.netlify.app)

## Features

- Sticky navigation bar with mobile hamburger menu
- Full-screen hero section with call-to-action buttons
- About section with restaurant story and stat badges
- Menu section with categorized dishes and prices
- Photo gallery
- Contact section with hours, location, and reservation form
- Fully responsive design (mobile & desktop)

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Sections

| Section | Description |
|---|---|
| Hero | Full-screen banner with tagline and CTA buttons |
| About | Restaurant story, founding history, and key stats |
| Menu | Appetizers, main dishes, desserts & drinks with prices |
| Gallery | Food photography grid |
| Contact | Address, hours, phone, email, and reservation form |

## Getting Started

To run locally, simply open `index.html` in your browser — no build tools or dependencies required.
